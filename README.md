# **Game of Life**

**demo in progress**

### **CZ Popis**

Simulace inspirovaná hrou Johna Conwaye. Používá mřížku (grid),
kde jsou buňky buď živé nebo mrtvé. Před spuštěním hráč označí
počáteční stav buněk, pak simulace běží podle čtyř pravidel:

1. Živá buňka s méně než 2 živými sousedy **zemře** (osamělost)
2. Živá buňka s 2 nebo 3 živými sousedy **přežije**
3. Živá buňka s více než 3 živými sousedy **zemře** (přelidnění)
4. Mrtvá buňka s přesně 3 živými sousedy **oživne** (reprodukce)

Projekt vznikl jako součást učení AI-assisted developmentu
s využitím GitHub Copilot pluginu ve VS Code.

### **EN Description**

A simulation inspired by John Conway's Game of Life. It uses a grid
where cells are either alive or dead. Before starting, the player
marks the initial state of the cells, then the simulation runs
according to four rules:

1. A live cell with fewer than 2 live neighbors **dies** (underpopulation)
2. A live cell with 2 or 3 live neighbors **survives**
3. A live cell with more than 3 live neighbors **dies** (overpopulation)
4. A dead cell with exactly 3 live neighbors **becomes alive** (reproduction)

This project was built as part of learning AI-assisted development
using the GitHub Copilot plugin in VS Code.
